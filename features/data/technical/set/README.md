# Technical Set
This feature is used to get all technical data

## Usage example
- method: ```POST```
- path: ```/api/data/technical```
- body: 
```json
{
    "technical": {
        "BBCA":{
            "160120304020":{
                "action": "Sell",
                "change": -0.77,
                "code": "BBCA",
                "date": "27 January 2022, 17:07 WIB",
                "id": "792,400",
                "name": "Bank Central Asia Tbk",
                "price": 7000,
                "status": "Strong Sell"
            }
        }
    }
}
```
